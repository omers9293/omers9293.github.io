Movie Webpage

This project is about putting what we know into a project that consists of everything we learned so far into the year. It was made with another partner and we had to plan it out using a markdown before we actually start the code. This helped us organize who is doing what and to keep us on task. The project is a quick story about how a pig was too "peppermint" or cool for a farm since he was in the rain when all the animaks werent. I purposely put a picture of a pig above the other animals to show how the pig was superior.

We had to use features like id, class, spans, borders, divs, paddings, etc. Some challenges that I faced was not being able to recall all the codes for the features that I had to use. For example, I supprisingly did not remember how to create a div so I can fix a spacing problem which got help from. If I had taken better notes, I could've reffered to my notes instead of my friend. A takeaway I learned is to take better notes so I can refer back to older information. To go more in depth for my challenges, I really struggled with correctly using the property margin which is why I tried to tinker it at jsbin.
