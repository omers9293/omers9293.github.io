Madlib CC Program

We learned how to use basic Javascript using concepts like <prompt()> and <document.write> that allowed us to question the viewers. We did this quick assignment with our partners and for me personally, there were no problems at all. I believe this was due to it being a quick 2 day assignment, instead of a long project that required more work to be put in. I did relearn that anything small like a + sign MATTERS. My <document.write> code did not work at all because I was missing a plus sign. This helped me become more aware of what I write and to go back and look for those small errors.

This was the first assignment I did using Javascript so I felt like it's something to be mentioned in my portfolio. Hopefully next time I will include a bigger project using Javascript.
